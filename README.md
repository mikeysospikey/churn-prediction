# Video Game Player Engagement Prediction

## Overview

This project explores machine learning approaches to modelling player engagement using a dataset of video game player behaviour. The aim is to understand which behavioural and demographic features are most predictive of engagement levels.

## Dataset 

The dataset which can be found [here](https://www.kaggle.com/datasets/rabieelkharoua/predict-online-gaming-behavior-dataset/data) includes the following features:

- PlayTimeHours 
- SessionsPerWeek
- AvgSessionDurationMinutes
- PlayerLevel 
- AchievementsUnlocked
- InGamePurchases
- Demographic features (Age, Gender, Location)
- Engagement (target varaible)

## Methods 

This project currently includes:

- Exploritory data analysis 
- Logistic regression modeling
